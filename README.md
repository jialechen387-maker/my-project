# R Simple Project

## Introduction
This is a simple R project created to practise basic data analysis and GitHub workflow.

## Project Structure
- data/: stores datasets
- scripts/: contains R scripts
- README.md: project description

## Methods
The project uses R for:
- creating a simple dataset
- generating summary statistics
- plotting basic relationships

## How to Run
1. Open the project in VS Code
2. Open `scripts/analysis.R`
3. Run the code line by line
